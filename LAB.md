# LAB

## Purpose

This document defines the local lab environment used to validate V1 features.

## Planned Lab Topology

- Development host: Ubuntu (WSL2)
- Linux target host over SSH
- Windows target host over SSH (OpenSSH enabled)

## Planned Validation Areas

- Dry-run behavior
- SSH command execution
- Branching on success/failure
- Retry and timeout handling
- Escalation notifications
- SQLite and audit log verification
